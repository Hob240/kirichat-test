## 1. Visi & Konsep Proyek

Kita sedang membangun sebuah SaaS AI Chatbot Builder yang canggih, bertujuan untuk bersaing langsung dengan layanan seperti chatbase.co.



Diferensiasi Utama (Killer Feature):

Bukan hanya sekadar chatbot, platform kita adalah sebuah Automation Hub. Nilai jual utamanya adalah Template Otomasi Industri siap pakai. User bisa memilih template (misal: Real Estate, Restoran, E-commerce) dan langsung mendapatkan chatbot yang terintegrasi dengan alur kerja bisnis yang relevan.



## 2. Arsitektur & Teknologi

Ini adalah tumpukan teknologi (tech stack) lengkap yang akan kita gunakan untuk mewujudkan proyek ini:



Frontend:



Framework: Next.js 14 stable (App Router)



Bahasa: TypeScript



Styling: Tailwind CSS



Backend - Fondasi & Pusat Komando:



Platform: Supabase

Fungsi:
Database (PostgreSQL): Menyimpan semua data user, agent, history chat, kontak, dll. (Supabase)


Authentication: Mengelola semua proses login, registrasi, dan manajemen user.


Storage: Menyimpan file mentah (PDF, TXT) yang di-upload oleh user.



Backend - Otak Pengetahuan (Proses Training AI):



Konsep: Retrieval-Augmented Generation (RAG).



Teknologi Inti: pgvector. Ekstensi di dalam database Supabase kita yang berfungsi sebagai Vector Database.



Proses: Ekstraksi Teks -> Chunking -> Embedding -> Penyimpanan di pgvector.



Backend - Otak Cerdas (Yang Menjawab Pertanyaan):



Teknologi: Large Language Model (LLM) API (Contoh: GPT-4o/GPT-5 dari OpenAI, Gemini dari Google).



Backend - Sutradara Otomasi (Yang Menjalankan Aksi):



Teknologi: Activepieces (saingan n8n tapi pake licensi MIT) (Self-hosted).



Fungsi: Menjalankan semua fitur di menu "Actions", template otomasi, dan integrasi kompleks.



Integrasi Pihak Ketiga (Kunci & Izin):



Teknologi: Aplikasi Kustom di Platform Developer (Contoh: Meta for Developers App).



Fungsi: Untuk mendapatkan API Keys & Kredensial Resmi yang akan digunakan oleh activepieces.
